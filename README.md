# Medicare Hospital Management System

A starter MERN-style hospital management system for admin, doctor, and patient workflows.

## Features
- Secure role-based authentication with JWT
- Admin dashboard with hospital summary
- Doctor dashboard with appointment handling
- Patient portal with appointment booking
- MongoDB-compatible models ready for production use

## Run locally

### Backend
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

### Frontend
```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```
